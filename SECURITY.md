# Security Policy

## Supported Versions

MindVault is early-stage. Security fixes are applied to the `main` branch.

## Reporting a Vulnerability

If you discover a security issue, please do **not** open a public issue.

Instead, send a private report with:

- A clear description of the vulnerability
- Steps to reproduce
- Impact assessment
- Suggested fix (if any)

Contact:

- **GitHub**: open a private advisory (Security → Advisories) if enabled
- Or open an issue with minimal details and ask maintainers to move it to private channels

We aim to respond within 7 days.
